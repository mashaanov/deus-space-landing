# Компонент Carousel (Карусель групп)

## 📋 Общее описание

Компонент реализует бесконечную карусель с автоматической прокруткой для отображения музыкальных групп. При клике на стрелки навигации автопрокрутка временно останавливается на 2 секунды, а при наведении на любую карточку — останавливается полностью до ухода мыши.

**Где используется:** Главная страница сайта

---

## 🎯 Основной функционал

| Функция | Описание |
|---------|----------|
| Бесконечная прокрутка | Слайды движутся непрерывно, без остановок и рывков |
| Автопрокрутка | Включена по умолчанию, скорость регулируется параметром `speed` |
| Остановка при наведении | При наведении мыши на любую карточку автопрокрутка останавливается |
| Стрелки навигации | При клике переключают на один слайд вперёд/назад, автопрокрутка возобновляется через 2 секунды |
| Пагинация (точки) | Показывает текущий слайд, клик переводит на выбранный слайд |
| Адаптивность | 1 слайд на мобильных, 2 на планшетах, 3 на десктопе |

---

## 🏗️ Используемые технологии

| Технология | Назначение |
|------------|------------|
| **Embla Carousel** | Базовый движок карусели (выбран вместо Swiper из-за отсутствия микропауз) |
| **Embla AutoScroll** | Плагин для непрерывной автоматической прокрутки |
| **React (hooks)** | `useState`, `useEffect`, `useCallback`, `useRef` |
| **Tailwind CSS** | Вся стилизация компонента |

## 🔧 Ключевые параметры настройки

### Конфигурация Embla Carousel

```javascript
{
  loop: true,           // бесконечный режим (позволяет прокручивать без конца)
  align: "start",       // слайды выравниваются по левому краю
  dragFree: true,       // можно свободно перетаскивать мышкой
  containScroll: false, // отключает ограничения прокрутки
  skipSnaps: false,     // плавная прокрутка, без "прилипания"
}
